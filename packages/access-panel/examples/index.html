Build de dist/web-component.js con las nuevas features: máscara de lectura, áreas grandes clicables y perfiles preset.

Texto y tipografía

Este párrafo existe para probar varios controles del panel: el tamaño de texto (4 niveles) debería escalar la fuente vía rem; el interlineado abre el aire entre líneas; el espaciado de letras separa caracteres; la fuente legible cambia a Arial y la fuente dislexia pasa a Comic Sans con espaciado extra.

La guía de lectura muestra una banda amarilla horizontal que sigue al cursor. La máscara de lectura oscurece toda la página dejando una "ventana" iluminada siguiendo al cursor — más restrictiva que la guía, útil cuando hay mucho ruido visual alrededor del texto.

Perfiles preset nuevo

Arriba del todo, los 4 botones de perfil disparan combinaciones de toggles:

  • Baja Visión → zoom + contraste + links destacados + cursor grande + outline foco
  • Dislexia → fuente dislexia + interlineado + espaciado + guía de lectura
  • Motor → cursor grande + áreas grandes + outline foco
  • Sin Distrac. → ocultar imágenes + pausar animación

Son aditivos — pulsar uno tras otro suma los flags. Para limpiar todo, el botón Restablecer Todo de abajo.
